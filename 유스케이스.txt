[사용자 시나리오]
1. 사용자는 자신이 찾는 직업(정보보안전문가)를 검색한다.
2. 직업 정보, 자격증, 공부 방법, 자격증 합격후기 경로 중에 1개를 선택한다.
3. 경로에 들어가 정보를 확인하고 뒤로가기 버튼을 통해 (2)의 화면으로 돌아간다.
4. 닫기 버튼을 통해 프로그램을 종료한다.

[기능 명세]
1. 검색 버튼의 동작 : 8개의 직업 데이터 중 하나를 검색
2. 직업 버튼의 동작 : 클릭 후 새로운 프레임으로 전환된다. (모든 직업 버튼이 정보보안전문가(3)으로 넘어간다.)
3. 직업 정보, 자격증, 공부 방법, 자격증 합격후기 버튼의 동작 : 새로운 프레임 전환된다.
4. 뒤로가기 : 직업 정보, 자격증, 공부 방법, 자격증 합격 후기 화면으로 돌아간다. (전 프레임 전환)
5. 닫기 : 프로그램 종료

[화면 구성]
초기 화면(시작 화면) : 모든 직업이 나온다 + 검색 기능, 닫기 기능
1.직업을 선택한 후 화면 : 직업 정보, 자격증, 공부 방법, 자격증 합격후기 버튼 + 뒤로가기 기능, 닫기 기능
1.1 작업을 선택한 후 화면 : 직업 정보, 자격증, 공부 방법, 자격증 합격후기 중 선택한 경로 내용 조회 + 뒤로가기 기능, 닫기
